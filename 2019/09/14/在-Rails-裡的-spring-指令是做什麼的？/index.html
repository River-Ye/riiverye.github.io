字數統計: 358閱讀時間: 1 min

自 Rails 4.1 以後，採用 Spring 來預載應用程式

保持應用程式在背景執行，如此一來在執行 Rails 命令時：如跑測試、rake、migrate 時，不用每次都重啟 Rails 應用程式，加速你的開發流程

特徵：

  • 自動化，毋須特別啟動或停止背景程序
  • 每次運行時會重新加載應用程序
  • 修改 configs / initializers / gem 時，建議重啟 server

相容性：

  • Ruby 版本：MRI 2.4 之後
  • Rails 版本：4.1 之後 (rails new 新專案時，預設安裝 Spring)
